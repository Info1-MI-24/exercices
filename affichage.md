# Affichage

Référence théorique : [handout](https://heig-tin-info.github.io/handout/content/stdio.html?highlight=printf#printf¨)

Quel est l'affichage **exact** des codes suivants ?
> les éspaces doivent être représentés

## Exercice 1
```c
int a = 5;
printf("%3d", a);
```

## Exercice 2
```c
double a = 3.1415;
printf("%6.2lf", a);
```

## Exercice 3
