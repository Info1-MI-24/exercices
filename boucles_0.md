# Exercices sur les boucles

Donner l'affichage **exact** pour chaque code ci-dessous.

## Exercice 1 :
```c
int i = 0;
while (i < 3) {
    printf("%d ", i);
    i++;
}
```

## Exercice 2 :
```c

    printf("%d,", i);
}
```

## Exercice 3 :
